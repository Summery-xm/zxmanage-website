# 更新日志
ZXManage 一直处于积极迭代中，致力于为开发者提供更高效的桌面端项目管理体验。以下是版本更新的详细记录。

## v1.0.0 <Badge type="tip" text="公测版" />

**发布日期: 2026-03-20**

### 🎉 全新公测版正式上线
ZXManage 1.0.0 公测版重磅发布！基于 Electron + React + Vite + TS 技术栈构建，聚焦「一站式本地项目全生命周期管理」，为开发者打造轻量化、高效率的桌面端项目管理工具。

### 🌟 核心功能全景
#### 📚 多技术栈项目统一管理
- 全面支持 Vue、React、Uni-app、Java、微信小程序、PHP、Node.js、Cocos 等主流技术栈项目
- 支持本地已有项目一键导入，自动识别项目git地址

#### 📂 本地文件与代码管理
- **文件管理模块**：支持文件/文件夹的新建、上传、打开、重命名、删除，可快速定位到系统文件资源管理器
- **代码管理模块**：可视化浏览项目代码目录树，支持文本文件在线查看/简易编辑，语法高亮，多文件标签页切换
- **Git 基础管理**：可视化查看文件 Git 状态，简化 Git 操作流程

#### 🚀 本地服务运行管控
- 基于项目配置自动生成服务列表，直观展示服务运行状态
- 支持服务的一键启动、重启、停止操作，实时显示服务访问地址
- 提前本地跑通打包后文件，规避服务部署后不好调试的问题

#### 📝 项目笔记管理
- 内置 Markdown 富文本编辑器，支持实时预览、语法高亮
- 笔记以 `.md` 文件形式存储到本地指定目录，支持创建、编辑、删除、搜索
- 笔记重命名自动保留原文件，生成新名称副本，保障数据安全

#### 💻 原生桌面端体验
- 响应式界面设计，操作流畅无卡顿
- 全功能本地运行，无需依赖云端服务，数据隐私有保障

### 📌 公测版说明
- 本次为首次公开测试版本，聚焦核心功能体验与稳定性验证
- 欢迎提交使用反馈，我们将持续优化功能细节与交互体验
- 目前版本不支持应用模块以及设置模块中的插件功能、快捷键功能
- 后续版本将迭代更多高级功能（如自定义快捷键、应用模块功能等）

### 🔧 技术栈
基于 Electron + React + Vite + TypeScript 构建，兼顾跨平台兼容性与开发效率。
